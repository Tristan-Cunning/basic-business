## Workflow Requirements

### Planning phase

- Planning should be completed prior to beginning the development phase and/or touching any code

### User stories

- Write out at least 3 user stories that describe how a user will interact with your site

### Wireframes

- Create wireframes of each page of site
  - Include important content and user interface elements like images, links, buttons etc.
- Can be done on paper or using widely available app's like draw.io

### Development phase

- Create a GitHub repository (Done)
- Clone it (Done)
- Make frequent descriptive commits throught dev.
